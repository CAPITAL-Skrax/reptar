    Contributing to Reptar

    Development

    Refer to DEVELOPMENT.md for docs on how to develop Reptar.

    Tests

    Tests are ran with mocha and written with ES2015+ JavaScript.

    Run tests via:

    npm test

    Linting

    We're using eslint for linting.

    Lint locally via:

    npm run lint
